# 一、Android动画介绍

# 1、基本概念

​ Android中动画分为帧动画(Frame Animation)、View 动画(View Animation)和属性动画(Property Animation)三类，帧动画是通过定义一组图片的播放顺序和时间来达到的一种动画效果，通常在页面加载时较为常用，View动画是通过定义起始和结束时动画的状态，以及期间的变换方式来达动画效果，属性动画通过改变View对象的属性值达到动画效果，这篇文章主要讲解View动画的基本用法和一个使用View Animation做的Demo，至于View Animation的绘制原理和Property Animation会在后面的文章中讲到。

# 2、View动画的四种类型:

​ 先来看一下View Animation动画中的四种类型：

类型 对应标签 对应类
平移动画 <translate> TranslationAnimation
缩放动画 <scale> ScaleAnimation
透明度动画 <alpha> AlphaAnimation
旋转动画 <rotate> RotateAnimation

# 3、四种动画类型的属性​

<translate>:

  • fromXDelta：开始时横坐标（比如50、50%、50%p）
  • fromYDelta：开始时纵坐标
  • toXDelta: 结束时横坐标
  • toYDelta: 结束时纵坐标

<scale>:

  • fromXScale: 动画开始时缩水平放比例（比如50、50%、50%p）
  • fromYScale: 动画结束时竖直方向缩放比例
  • toXScale: 动画结束时水平方向缩放值
  • toYScale: 动画结束时竖直方向缩放值
  • pivotX: 缩放轴点x坐标
  • pivotY: 缩放轴点y坐标

<rotate>:

  • fromDegrees: 旋转开始时角度
  • toDegrees: 旋转结束时角度
  • pivotX: 旋转轴点x坐标
  • pivotY: 旋转轴点y坐标

<alpha>:

  • fromAlpha: 动画开始时透明度
  • toAlppha: 动画结束时透明度

通用属性：

  • duration:动画持续时间，单位ms
  • fillAfter:[true|false] 动画结束以后View是否保留在结束位置。
  • fillBefore:[true|false] 动画结束时View是否返回到开始位置
  • fillEnabled:[true|false]    与android:fillBefore 效果相同，都是在动画结束时，将控件还原到初始化状态
  • repeatMode:[reverse|restart] 重复模式，reverse表示倒序回放，restart表示从头重放
  • repeatCount:[] 动画重复次数
  • interpolator：设置插值器

# 4、两种定义动画的方式

# (1) 使用XML定义动画：

anim_test.xml:

<?xml version="1.0" encoding="utf-8"?>
<alpha xmlns:android="http://schemas.android.com/apk/res/android"
    android:duration="600"
    android:fromAlpha="0.0"
    android:toAlpha="1.0" />

​ 上面的xml文件定义了一个透明度渐变动画，效果是在600ms的时间里控件的透明度从0变成1，也就是从完全透明变成完全不透明。要在某个控件上使用这个动画效果也很简单，先使用系统提供的AnimationUtils类里的加载方法加载xml文件，然后调用View的startAnimation启动动画就行了。

Animation animation = AnimationUtils.loadAnimation(this,R.anim.anim_text);
textView.startAnimation(animation);

# (2) 使用代码定义动画：

​ 下面是使用代码的方式来定义动画，效果是在1秒的时间里大小从0放大到原来的大小。虽然用代码也能完成任何xml的效果，但是当动画比较复杂时使用代码远不如用xml直观易懂，所以一般情况下是使用第一种方式来定义动画的。

ScaleAnimation scaleAnimation = new ScaleAnimation(0,1,0,1);
scaleAnimation.setDuration(1000);
textView.setAnimation(scaleAnimation);

# 二、实例

# 1、每种动画单独使用

  • # TranslateAnimation动画

    效果：向右下角移动

<?xml version="1.0" encoding="utf-8"?>
<translate xmlns:android="http://schemas.android.com/apk/res/android"
    android:duration="600"
    android:fromXDelta="0"
    android:fromYDelta="0"
    android:toXDelta="100"
    android:toYDelta="100"
    android:fillAfter="true"
    android:interpolator="@android:anim/linear_interpolator"
    />

  • # ScaleAnimation动画

    效果：大小从0缩放到1

<?xml version="1.0" encoding="utf-8"?>
<scale xmlns:android="http://schemas.android.com/apk/res/android"
    android:duration="600"
    android:fromXScale="0"
    android:toXScale="1"
    android:fromYScale="0"
    android:toYScale="1"
    android:interpolator="@android:anim/anticipate_overshoot_interpolator"
    />

  • # AlphaAnimation动画

    效果：透明度从0到1

<?xml version="1.0" encoding="utf-8"?>
<alpha xmlns:android="http://schemas.android.com/apk/res/android"
    android:duration="600"
    android:fromAlpha="0"
    android:toAlpha="1"
    android:interpolator="@android:anim/anticipate_overshoot_interpolator"
    />

  • # RotateAnimation动画

    效果：顺时针旋转90度

<?xml version="1.0" encoding="utf-8"?>
<rotate xmlns:android="http://schemas.android.com/apk/res/android"
    android:duration="600"
    android:fromDegrees="0"
    android:toDegrees="180"
    android:fillAfter="true"
    android:interpolator="@android:anim/linear_interpolator"
    />

# 2、使用<set>集合动画

​ set集合可以将多个动画连续运行，下面的xml的效果是控件在向右下角移动的过程中顺时针旋转90度。

<?xml version="1.0" encoding="utf-8"?>
<set  xmlns:android="http://schemas.android.com/apk/res/android"
    android:fillAfter="true"
    android:repeatMode="restart"
    >
    <translate
        android:duration="600"
        android:fromXDelta="0"
        android:fromYDelta="0"
        android:toXDelta="100"
        android:toYDelta="100"
        />
    <rotate
        android:duration="600"
        android:fromDegrees="0"
        android:toDegrees="10"
        />
</set>
