越来越多的人使用hexo来写博客，简洁又美观，是在是很符合我的喜好，前几天看了下如何搭建后，今天终于腾出了些时间来搭建一个hexo博客，今后便放弃使用wordpress了。搭建过程很简单，出于习惯还是记录一下。

# 首先安装nodejs和hexo：

	#node安装命令
	brew install node  #如果我没记错的话,最新版的node.js的包中已经集成了npm包管理工
	#具
	#使用以下命令验证是否安装成功
	node -v
	npm -v
	npm install hexo -g  #安装hexo
	hexo init <folder> #初始化hexo到指定目录
	hexo gennerate #根据安装目录下文件生成静态网页
	hexo server #运行hexo服务

此时在http://localhost:4000已经能看到hexo的效果

# hexo的使用：

	hexo new "postName" #新建一个名为postName的文章
	此处及以上的内容会在首页显示
	<!--more-->
    以下是在首页隐藏的部分
    #修改主题
    git clone git@github.com:yunlzheng/hexo-  themes.git themes/writing
    #在./_config.yml，修改主题为writing
    theme: writing
    #查看本地效果
    hexo g       #hexo generate简写
    hexo s       #hexo server简写

# 部署hexo到github：

1.打开安装文件夹中的——config.yml文件

	#修改博客信息后在文件末尾添加git信息如下
	# Deployment
	##Docs: https://hexo.io/docs/deployment.html
	deploy:
    	type: git  #这里填写git
        repository: git@github.com:ZacharyWZH/  zacharywzh.github.io.git #远端仓库地址
        branch: master	  #部署到的分支

2.部署到github的命令

hexo deploy

# 使用过程中遇到的问题和解决方法：

# 1.点击标签页跳转404：

解决方法：

尝试重新生成页面 hexo new page tags

然后在title，date下面加上type: "tags"

如果不希望评论，还可以加上：comments: false
# 2.升级hexo版本或更改目录重新安装后：ERROR Deployer not found: git
npm install hexo-deployer-git --save
