# #1. Two Sum

Problem:

Given an array of integers, return indices of the two numbers such that they add up to a specific target.
You may assume that each input would have exactly one solution.
Example:
Given nums = [2, 7, 11, 15], target = 9,
Because nums[0] + nums[1] = 2 + 7 = 9,
return [0, 1].
UPDATE (2016/2/13):
The return format had been changed to zero-based indices. Please read the above updated description carefully.

这道题就是应用了HashMap，比较简单。

Solution:

public static int[] Solution(int[] nums, int target) {
        int[] result = new int[2];
        HashMap<Integer,Integer> map = new HashMap<Integer,Integer>();
        for(int i=0;i<nums.length;i++){
        	if(map.containsKey(target-nums[i])){
        		result[1] = i+1;
        		result[0] = map.get(target-nums[i]);
        	}
        	map.put(nums[i], i+1);
        }
        return result;
    }

# #2. Add Two Numbers

Problem:

You are given two linked lists representing two non-negative numbers. The digits are stored in reverse order and each of their nodes contain a single digit. Add the two numbers and return it as a linked list.
Input: (2 -> 4 -> 3) + (5 -> 6 -> 4)
Output: 7 -> 0 -> 8

这道题比较思路比较精妙，难度稍大，不仅是ListNode对应相加，还涉及进位的问题，参考网上的答案之后解决如下，增加了几行注释。

Solution:

public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        int c = 0;                //进位数
        ListNode result = null;
        ListNode head = null;
        while(l1!=null||l2!=null||c!=0){
            int l1Val = l1 == null ? 0 : l1.val;
            int l2Val = l2 == null ? 0 : l2.val;
            int sum = l1Val + l2Val;
            int cur = (sum+c)%10;             //当前位的数字
            c = (sum+c)/10;		             //要进位的数字等与当前位的和加上上一位进过来的数字除以10
            if(result != null){
               result.next = new ListNode(cur);
               result = result.next;
            }else{ 							//第一次循环时result为null
                result = new ListNode(cur);
                head = result;
            }
            if(l1 != null) l1 = l1.next;
            if(l2 != null) l2 = l2.next;
        }
        return head;

    }
